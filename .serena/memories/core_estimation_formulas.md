## 핵심 공식 (ai_estimation_core_v1.2.0.json)

### PVC 커버 비용 (철재 외함용)
```
공식: (W × H) / 90000 × 12000 KRW
단위: 원 (100원 단위 반올림)
```

### 도장 면적 계산
```
공식: 2 × (W×H + W×D + H×D)
단위: mm² → m² 변환 (× 0.000001)
```

### 공간 활용률
```
공식: used_volume / total_enclosure_volume
최대: 0.85 (안전 여유 0.15)
```
